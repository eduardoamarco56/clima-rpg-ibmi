# Screenshots

Esta carpeta contiene las capturas de pantalla de la aplicación en funcionamiento.

## Cómo agregar capturas de pantalla:

1. **Toma capturas de pantalla de tu aplicación:**
   - Página principal con el formulario de búsqueda
   - Resultados mostrando el clima de una ciudad
   - Vista en diferentes navegadores (opcional)

2. **Guarda las imágenes aquí con nombres descriptivos:**
   - `home.png` - Página principal
   - `search-result.png` - Resultado de búsqueda
   - `mobile-view.png` - Vista móvil (opcional)

3. **Formatos recomendados:**
   - PNG para capturas de pantalla (mejor calidad)
   - JPG para fotos (menor tamaño)
   - Tamaño recomendado: máximo 1920x1080px

## Ejemplo de nombres de archivo:

```
docs/screenshots/
├── home.png
├── search-result.png
├── weather-details.png
└── mobile-view.png
```

Una vez que agregues las imágenes aquí, se mostrarán automáticamente en el README principal del proyecto.
